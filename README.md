# Never Alone Client

Modern Next.js 16 frontend for the Never Alone platform. Includes public landing, event discovery, and admin dashboards for managing carousels and events.

## Tech Stack

-   Next.js 16 (App Router)
-   React 19, TypeScript
-   TailwindCSS 4
-   Radix UI primitives + custom UI kit
-   Embla carousel for interactive sliders

## Features

-   Public landing with hero, carousel banners, and top events.
-   Event listings and detail navigation (links to `/events` and `/events/[id]`).
-   Admin dashboards for carousels (create/delete) and categories.
-   Reusable UI components (cards, buttons, dialogs, table, carousel).
